## Позитивный тест-кейс:
### Форма отправляется только при заполненном поле Name
**Предусловие:**

1. Открыть браузер
2. Перейти по ссылке https://practice-automation.com/form-fields/

**Шаги:**
    
1. В поле "Name" ввести "Валидное имя"
2. Проскроллить страницу вниз
3. Нажать на кнопку "Submit"

**Ожидаемый результат:**
Появляется алерт с сообщением "Message received!"
****

## Негативный тест-кейс: 
### При попытке отправить форму с пустым полем Name срабатывает валидация
**Предусловие:**

1. Открыть браузер
2. Перейти по ссылке https://practice-automation.com/form-fields/

**Шаги:**

1. Проскроллить страницу вниз
2. Нажать на кнопку "Submit"

**Ожидаемый результат:**
1.Страница автоматически проскроллится наверх 
2.Каретка будет автоматически установлена на поле "Name"
